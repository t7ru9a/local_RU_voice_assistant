# Voice AI Assistant (Whisper + LM Studio + Piper)
Простой голосовой ассистент на Python. Позволяет общаться с локальной языковой моделью голосом: вы нажимаете клавишу, задаете вопрос, а нейросеть отвечает вам через синтез речи.
## ✨ Особенности
STT (Голос в текст): Использование faster-whisper для мгновенного распознавания.
LLM (Интеллект): Интеграция с LM Studio (поддерживает любые Llama, Mistral, Gemma модели).
TTS (Текст в голос): Озвучка через быстрый локальный движок Piper.
Конфиденциальность: Всё работает на вашем компьютере, данные не отправляются в облако.
## 🛠️ Требования
Python 3.10+
LM Studio.
Piper: Утилита piper.exe должна быть установлена и добавлена в системную переменную PATH.
CUDA (Опционально): Для ускорения Whisper на видеокартах NVIDIA.
## 🚀 Быстрый старт
1. Клонирование и установка
bash
git clone https://github.com
cd voice-assistant-py
pip install -r requirements.txt
Используйте код с осторожностью.
2. Настройка голосов
Поместите файлы моделей Piper (.onnx и .json) в папку:
resources/piper-voices/
Пример структуры:
resources/piper-voices/ruslan.onnx
resources/piper-voices/ruslan.onnx.json
3. Настройка путей (config.py)
Откройте config.py и проверьте пути к библиотекам CUDA (если используете GPU) и названия файлов голосов.
4. Запуск
bash
python main.py
Используйте код с осторожностью.
## 🎮 Как пользоваться
Запустите LM Studio, выберите модель.
Запустите скрипт main.py.
Зажмите клавишу Alt и говорите.
Отпустите клавишу — ассистент обработает запрос и ответит голосом.
## 📂 Структура проекта
main.py — точка входа, логика управления.
audio_handler.py — запись звука с микрофона.
ai_logic.py — взаимодействие с Whisper и LM Studio.
config.py — все настройки в одном месте.
resources/ — папка для моделей синтеза речи.
